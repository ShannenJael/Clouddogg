# CloudDogg Next.js Project

This is a VS Code-ready Next.js project scaffold based on the homepage rebuild.

## Open in VS Code
1. Download and extract the zip.
2. Open the `clouddogg-nextjs-vscode` folder in Visual Studio Code.
3. Run:

```bash
npm install
npm run dev
```

4. Browse to `http://localhost:3000`

## Run In Docker
Build and start the containerized app with:

```bash
docker compose up --build -d
```

Then browse to `http://localhost:8084`.
